# 💰 Expense Tracker Pro (AI-Powered FinTech Dashboard)

🚀 A complete **Data Science + Machine Learning + Interactive Dashboard Project** that helps users track, analyze, and predict their expenses using a modern, professional UI.

---

## 📌 Project Overview

Expense Tracker Pro is an **AI-powered financial analytics system** built to simulate real-world FinTech applications like Google Pay, CRED, and Mint.

It allows users to:

* Track daily expenses
* Analyze spending behavior
* Visualize trends using charts
* Get intelligent insights
* Predict future expenses using Machine Learning

---

## 🎯 Key Features

### 🔐 Authentication System

* Secure login interface
* Session-based authentication using Streamlit

### 📊 Interactive Dashboard

* Category-wise expense analysis
* Monthly trend visualization
* Pie chart distribution
* Top transactions table

### 🎨 Modern UI/UX

* Glassmorphism design
* Animated gradient background
* Dark / Light mode toggle
* Clean fintech-style layout

### 💡 Smart Insights

* Detects highest spending category
* Calculates average transaction
* Flags overspending behavior

### 🚨 Budget Tracker

* Set monthly budget
* Real-time budget alerts

### 📄 Report Generation

* Download expense data as CSV

## 🚀 How to Run the Project

### 1️⃣ Clone Repository

git clone https://github.com/yourusername/Expense-Tracker-Pro.git
cd Expense-Tracker-Pro

### 2️⃣ Install Dependencies

pip install -r requirements.txt

### 3️⃣ Run Data Pipeline

python src/data_pipeline.py

### 4️⃣ Train ML Model

python src/spending_model.py

### 5️⃣ Launch Dashboard

streamlit run app/app.py
